Interspect

An open-source API-mocking tool to help ensure data interoperability between microservices without having to worry about test data set up or environment availability. Interspect is specifically designed to maximize developer and tester productivity when working in autonomous or cross-functional product teams

Quick mockup

Modify data from a source microservice to creat mockups for easy testing. Unblocking testers and developers waiting for APIs

Batch testing

Send HTTP request for each mockup in one click to speed up your work flow. Reduce third-party transaction cost

Performance

View network timming metrics for your tests to catch latency issues upfront. Make your performance tests more reliable.
